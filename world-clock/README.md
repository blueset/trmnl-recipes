## Parameters

- Time format:
  - 12-hour (midnight starting at 12:00 AM)
  - 12-hour (midnight starting at 0:00 AM)
  - 24-hour
- Home Time Zone (default to user time zone)
- Home name, default to “Home”
- Additional time zones (up to six), and names
