Tailwind Play CDN for TRMNL

An in-browser Tailwind CSS v4 compiler designed for TRMNL recipe development. It processes <style type="text/tailwindcss"> blocks on the fly, so you can use Tailwind utility classes directly in your Liquid templates without a build step.

Hosted files

  • trmnl-play-cdn.mjs — the browser-side Tailwind compiler module
  • trmnl-shared.css — shared TRMNL theme with custom variants and utilities

Quick start

Add the following to your recipe’s Shared section:

<script type="module" src="https://blueset.github.io/trmnl-recipes/trmnl-play-cdn.mjs"></script>
<style type="text/tailwindcss">
@import "trmnl-shared.css";
</style>

The script picks up every <style type="text/tailwindcss"> block in the page, compiles the CSS with Tailwind v4, and injects the result into the document. It also watches the DOM for class-attribute changes and new elements, triggering incremental rebuilds automatically.

Custom variants

trmnl-shared.css defines the following TRMNL-specific variants:

  • Screen size: lg, md
  • Color depth: pix (1-bit & 2-bit), 1bit, 2bit, 4bit
  • View type: full, hori, vert, quad
  • Orientation: land, port

Custom utilities

Typography utilities matching TRMNL’s design tokens:

  • content-{small,base,large,xlarge,xxlarge,xxxlarge}
  • description-{base,large,xlarge}
